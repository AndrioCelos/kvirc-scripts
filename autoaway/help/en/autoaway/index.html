Now Playing : The Media Advertiser

The "Now Playing" addon allows you to advertise the media you're currently playing on IRC channels and queries.

The media players supported are the ones supported by the mediaplayer kvirc module: at the moment of writing "xmms", "amarok", "juk" and "winamp".

How it works

You advertise your media by the means of a toolbar button (that be visible in your KVIrc just after the installation) or by the means of the nowplaying_advertise command. This command in fact uses /me which in turn sends a CTCP ACTION to the target (chanel or query).

The message displays the title and the artist of the played media and a set of optional informations including bitrate, file size etc. The format of the message can be configured by accessing proper option in the Script Addon Management dialog or by using the nowplaying_configure command.

The advertised media can be optionally shared with other users that can request the download by sending you a simple CTCP command.
