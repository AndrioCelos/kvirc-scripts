Twins Auto Op

An auto op, voice and ban system

Author:Ahinu

ahinu.laman@gmail.com

How it works

This addon is not only an auto op system, it provides also auto voice and auto ban. It takes advantage of the $reguser and $regchan modules: for each channel record (defined by a channel name and a netmask) and action (op, voice, ban), a list of registered users is defined. If such user (identified by his masks) join such channel, he will be automatically oped, voiced or banned. The addons checks for actions to perfom on users both when a user join the channel or when get oped.

The auto op/voice is performed after a specified delay (default is 10s) for security reasons. A user can be oped and voiced at the same time.

The auto ban system is nothing else but a "blacklinsting system". If a registered user is blacklisted on a channel, he will be kick/banned according to his matched registered mask. Note: if banned, this user won't be auto oped/voiced.

Configuration

Here is a list of what you can configure:

  • Manage the list of registered users for each action on each channel.
  • Unregistered channels and users can be registered directly using the addon.
  • Actions can be suspended independantly on each channel.
  • You can set the delay before auto op/voice, and the default kick reason for auto bans.
You can configure the addon in different ways:
  • From the addon manager or using twins::autoop::configure, you open the main configuration dialog.
  • On channels, you can right-click in the main window or on nicknames to open specific popups.
  • Specific settings are accessible using below commands.

      Commands

      twins::autoop::configure
      Open the main configuration dialog.
      twins::autoop::help
      Show help of the Twins AutoOp addon
      twins::autoop::set::user [--help] [<user>]
      Set or unset an auto action for a specified user on a channel.
      twins::autoop::set::chan [--help] [--op/-o[=0/1]] [--voice/-v[=0/1]] [--ban/-b[=0/1]] [<chan>[@<network>]
      Activate/deactivate auto actions or show status on a channel
      twins::autoop::set::delay [--help] [<delay:seconds>]
      Show or set the delay for auto op/voice after join
      twins::autoop::set::kickreason [--help] [<reason:string>]
      Show or set the kick reason for auto kick/bans
      twins::autoop::set::recheck [--help] [--op/-o[=0/1]] [--voice/-v[=0/1]] [--ban/-b[=0/1]]
      Set or show status for rechecking
      twins::autoop::check [--help] [-q] [--op/-o[=0/1]] [--voice/-v[=0/1]] [--ban/-b[=0/1]] [<nick>]
      Check if auto actions must be performed on the current channel
